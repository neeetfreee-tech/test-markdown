- `**текст**` или `__текст__` — **Жирный**
- `~~текст~~` — ~~Зачеркнутый~~
- `==текст==` — ==Маркер / Подсветка==
- `` `текст` `` — `Внутристрочный код`
- `\*текст\*` — Экранирование спец символов, экран = \*текст\*  без = *текст*
